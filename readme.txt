# This is a pre-build kernel installer package.
# For configuring your package, check `!zygote.sh` or gearlock can also guide you during the build process.

# Put the `kernel` image file in the rootdir of this package.

# Put the `kernel module dir` into `system/lib/modules`

# If you want to also include `custom firmware` alongside the `kernel modules` then put `firmware` folder into `system/lib`
# Only include firmware if you feel it's necessary or if you want to.
